Projetos Independentes no Mercado

APLICATIVO EDUK ALUNO

Desenvolvimento de um aplicativo que se integra ao sistema web das faculdades para uso dos alunos.
O aplicativo permite facilitar aos alunos a geração boleto, geração 2ª via de boleto, compatilhameno de arquivos, armazenamento de arquivos no dispositivo. Além do acesso de notas, também possível alterar informações pessoais do próprio aluno.
► Desenvolvido em Flutter e Laravel (API Rest)
► Persistência de dados;
► Uso online
► Manipulação de dados via API Rest
► Disponível na Play Store e Apple Store
